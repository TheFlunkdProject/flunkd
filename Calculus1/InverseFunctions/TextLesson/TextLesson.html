1.1 - Graphs of Basic Functions

The three things in this section are:
  • How to determine if something is a function
  • How to determine if a function is one-to-one
  • How to find inverse functions

1. How to determine if something is a function:

is a function if no value of x has two corresponding y values.So if you plug in some number for x, there will either be no y value that pops out, or exactly one. Most of the things we deal with in Calculus 1 are functions, because they are useful and easy to work with.

If you know what the graph of f(x) looks like, you can apply the vertical line test. If you can draw a vertical line anywhere on the graph, and the curve f only crosses it once or never, f is a function.The vertical line crosses the function only once, so this is a function. If there is any place where f crosses a vertical line twice, then f is not a function.Circles fail the vertical line test. The line crosses the circle twice, so circles are not functions.

2. How to determine if a function is one-to-one:

A function is one-to-one if the same test for a function is applied the other way around: When you plug in a y value, either one or no value comes out for x.

If you know the graph of a function, it is often easier to draw the graph and apply the horizontal line test than to solve for x. This is similar to the vertical line test, but instead of drawing a vertical line, you draw a horizontal line across the graph. If it crosses once or never, it is one-to-one.

3. How to find inverse functions:

There are three steps to do this.
  1. Switch all the x's and y's
  2. Solve for the new y. (Get the y by itself on one side of the equation. This is the actual inversion process.)
  3. Sometimes it may be necessary to change the y out for a f-1(x). This signifies that it is the inverse of f(x). f(x) and y are generally the same.
    (Remember, though, that f-1(x) is not the same as having a negative 1 exponent.)
    To find what the inverse of a graph looks like, just flip the graph over the line y=x. You can also take each point in the original function and switch the x and y coordinates.
